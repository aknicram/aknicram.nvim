# sumptuous.nvim

A dark color scheme for neovim.

Mix of grayscale and [pastel colors]([https://colorkit.co/color/d77991).

## Installation

```lua
-- Lazy
{ "aknicram/sumptuous.nvim" }
```

```lua
vim.cmd("colorscheme sumptuous")
```
